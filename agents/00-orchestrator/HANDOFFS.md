# Handoff Log

에이전트 간 작업 인계 기록. 최신 항목이 위에 오도록 추가.

---

## Template (복사해서 사용)

```markdown
## YYYY-MM-DD — [Task title]

| Field | Value |
|-------|-------|
| **From** | 00-orchestrator |
| **To** | 01-ops-sre |
| **Priority** | P0 / P1 / P2 / P3 |
| **Goal** | 한 줄 목표 |
| **Context** | 관련 파일/링크 |
| **Acceptance** | 완료 조건 (체크리스트) |
| **Out of scope** | 하지 말 것 |
| **Blocked by** | 없음 / 에이전트명 |
```
